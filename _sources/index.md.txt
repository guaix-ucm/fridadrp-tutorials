# FRIDA DRP tutorials

This webpage contains a first draft of the documentation on the use of the data
reduction pipeline of the FRIDA instrument.

[FRIDA](https://www.gtc.iac.es/instruments/frida/frida.php) (in**FR**ared
**I**mager and **D**issector for **A**daptive optics) is an Integral Field
Spectrograph that will make use of the GTC [Adaptive Optics system
(GTCAO)](https://www.iac.es/en/projects/gtcao-lgs-adaptative-optics-and-laser-guide-star-gtc/).

```{warning}
As the software is under development, it is expected that significant
modifications will be introduced to the procedures described below.
```
